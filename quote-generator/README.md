Суть: Создать страницу с блоком для текста и кнопкой. При нажатии на кнопку в блоке должна появляться случайная цитата.
Технические требования (все выполнены):
	•	Массив объектов  quotes  — каждый объект имеет свойства  text  и  author 
	•	Функция случайного выбора — выбирает случайный объект из массива
	•	DOM манипуляции — обновление  textContent  соответствующих элементов
	•	Мини-вызов: Цитаты не повторяются два раза подряд ( do...while )

Быстрый старт

# 1. Создай Vite проект
npm create vite@latest quote-generator -- --template vanilla
cd quote-generator
npm install

# 2. Замени файлы:
#   - index.html (корень)
#   - main.js (корень) 
#   - src/style.css
#   - README.md (этот файл)

# 3. Запуск
npm run dev
