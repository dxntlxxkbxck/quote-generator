Цель: Работа с массивами и простыми манипуляциями DOM.

Суть: Создать страницу с блоком для текста и кнопкой. При нажатии на кнопку в блоке должна появляться случайная цитата.

Технические требования:

Создать массив объектов quotes, где каждый объект имеет свойства text и author.

Написать функцию, которая выбирает случайный объект из массива.

При клике на кнопку обновлять textContent соответствующих элементов в HTML.

Мини-вызов: Сделайте так, чтобы цитаты не повторялись два раза подряд.
